# 🏠 House Price Prediction Using Linear Regression

## 📌 Project Overview

This project is a Machine Learning application that predicts house prices using the Linear Regression algorithm. It demonstrates the complete machine learning workflow, including data preprocessing, feature selection, model training, evaluation, and prediction using the Kaggle House Prices dataset.

## 🎯 Objective

Predict the selling price of a house based on important features such as:

* Lot Area
* Overall Quality
* Overall Condition
* Year Built
* Ground Living Area
* Number of Bedrooms
* Number of Bathrooms
* Garage Capacity
* Garage Area

## 📂 Dataset

This project uses the **Kaggle House Prices - Advanced Regression Techniques** dataset.

Files included:

* `train.csv` – Used to train and evaluate the machine learning model.
* `test.csv` – Contains unseen house data for making predictions.
* `sample_submission.csv` – Provides the required submission format for Kaggle competitions.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## 📈 Machine Learning Workflow

* Load the dataset
* Explore and understand the data
* Handle missing values
* Select important features
* Normalize numerical features using StandardScaler
* Split the training data into training and testing sets
* Train a Linear Regression model
* Evaluate the model using:

  * Mean Squared Error (MSE)
  * R² Score
* Predict house prices
* Visualize Actual vs Predicted house prices

## 📊 Evaluation Metrics

* Mean Squared Error (MSE)
* R² Score
